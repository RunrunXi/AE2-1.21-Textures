# AE2 1.21 材质包

这是一个适用于 Minecraft 1.20+ 版本的资源包，将 Applied Energistics 2 (AE2) 模组的材质和模型从 1.20 版本风格更新为 1.21 版本风格。

## ✨ 主要内容

- **新材质**：将 AE2 模组的所有方块、物品更新为 Minecraft 1.21 风格的材质

## 🚀 安装方法

1. 下载最新的 `.zip` 文件
2. 将文件放入 Minecraft 的资源包文件夹：
   - Windows: `%appdata%\.minecraft\resourcepacks\`
3. 在游戏中选择选项 → 资源包 → 启用本材质包

## 🎨 更新内容

### 已更新的材质
- ✅ [AE2]应用能源2 (Applied Energistics 2)
- ✅ [AAE] 高级AE (AdvancedAE)
- ✅ [EAE] AE2扩展 (ExtendedAE)

### 计划更新
- 🔄 AE2 Things
- 🔄 AE2 MEGA Things

## 📝 注意事项

- 本材质包**仅修改视觉效果**，不改变任何游戏机制
- 确保 AE2 模组已正确安装
- 如果遇到材质显示问题，请检查资源包加载顺序
